## Machine Learning LifeCycle Stages as follows

  Collect Data
  Process Data - 
    Features and Lables
    Feature Engineering
    Feature Reduction
    Encoding
    Formatting

  Split Data
    Split into _Training_, _Validataion_, & _Test_ Data
  Train 
  Test
  Deploy
  Infer   (from Infer, if required data will be send to _Split_ stage for improvement)
  make the **Predictions**
